# Tripleten web_project_homeland
# De Patria a patria

## Descripción
Proyecto de pagina web responsiva con respecto al Sprint 5. 
El sitio está optimizado para dispositivos móviles, tablets y escritorio.

## Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox
- Grid CSS
- Metodología BEM
- Diseño responsivo

## Enlace al proyecto
https://github.com/bruno2017-href/web_project_homeland

## Funcionalidades
- Diseño responsivo para 1280px - 768px - 320px
- Navegación fluida

## Instalación
1. Clona este repositorio
2. Abre index.html en tu navegador
